NotepadX


NotepadX is a very simple and straight forward software with 4 basic functionalities of Create New File, Save File, Load any text files from computer and quit application. Most shortcut keys such as (Ctrl + C, Ctrl + V, etc.) work in notepadx. There are also validations in case a file couldn't be saved or loaded, users can also track the current state of the application from the status bar on the right side.
